webpack常用插件

2018-09-05

1. html-webpack-plugin

主要作用：

  1. 为html文件中引入的外部资源如script、link动态添加每次compile后的hash，防止引用缓存的外部文件问题
  2. 可以生成创建html入口文件，比如单页面可以生成一个html文件入口，配置N个html-webpack-plugin可以生成N个页面入口

配置项：

  1. title:设置生成的 html 文件的标题
  2. filename:生成 html 文件的文件名。默认为 index.html
  3. template:根据指定的模板文件来生成特定的 html 文件。模板类型可以是 html, jade, ejs 等等（使用自定义模板时应该安装对应的loader，否则webpack不能正确解析）

以jade为例：

如果既指定了 template 选项，又指定了 title 选项，webpack 会选择你指定的模板文件的 title, 即使你的模板文件中未设置 title。
filename也不会被覆盖，会以指定的 filename 作为文件名。
